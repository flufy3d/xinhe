# 🧠 心核 架构蓝图 终极整合版：大一统快慢交替记忆网络
*(Grand Unified Fast-Slow Alternating Memory Network w/ Task-Explicit Dual-Track MoE)*

这是一份将 v2.0 的动态演化细节与 v3.0 的物理硬隔离机制完美融合的最终蓝图。新架构保留了基于余弦增量的离线人格注入协议，并彻底废弃了不可控的动态路由，采用**任务显式双轨新皮层 (Task-Explicit Dual-Track MoE)**，从数学和物理层面实现了人格底色与日常阅历的绝对隔离。

---

### 🌌 核心设计哲学

**"统一海马体，双轨新皮层；物理隔离固底色，动静交替刻流年。"**

模型由三层记忆载体协同运作：
1. **永久冻结的 Base 基座：** 承载预训练的通用世界知识与语言法则，提供底层智力。
2. **全局唯一的海马体 W（Attention 侧）：** 承载当日工作与情景记忆。依靠 Delta Rule 按内容寻址精准覆写，快频吸收眼前的新鲜事务。
3. **并联的双轨新皮层（Task-Explicit MoE）：** 彻底废弃动态路由机制。将新增的神经元物理硬分割为**「人格轨 (Persona Expert)」**与**「阅历轨 (Episodic Expert)」**。根据当前所处的生命周期阶段，由系统强制指定激活与更新的轨道。

记忆的生命周期被划分为**胚胎期（离线人格注入）**、**白昼（快感官体验）**与**黑夜（慢蒸馏固化）**。

### 🧮 核心数学原理：海马体与无路由双轨新皮层

#### 1. 海马体：大一统状态演化方程 (纯 Delta Rule)
在 Attention 侧，单一状态张量 $W$ 的演化完全抛弃基于时间的衰减，采用纯几何的正交槽位覆写：
$$W^{(t)} = W^{(t-1)} + \beta_t \cdot (v_t - W^{(t-1)}k_t) \otimes k_t^T$$
读侧以常数复杂度匹配，其中 $q_t, k_t$ 严格做 L2 归一化：
$$\text{read} = q_t \cdot W^{(t)\,T}$$

#### 2. 新皮层：无路由的双轨加法结构
在每一层 Transformer Block 中，并联的新皮层退化为极其稳定、无路由参数的并行加法结构：
$$\text{MLP}_{out}^{(\ell)}(x) = \underbrace{\text{MLP}_{base}^{(\ell)}(x)}_{\text{永远冻结}} + \underbrace{\mathbb{I}_{p} \cdot \text{Expert}_{persona}^{(\ell)}(x)}_{\text{人格轨}} + \underbrace{\mathbb{I}_{e} \cdot \text{Expert}_{episodic}^{(\ell)}(x)}_{\text{阅历轨}}$$
* **$\text{Expert}_{persona}$ 与 $\text{Expert}_{episodic}$**：均为独立的全秩 SwiGLU 网络。
* **$\mathbb{I}_{p}, \mathbb{I}_{e} \in \{0, 1\}$**：这是由当前系统所处**生命周期**直接给定的布尔控制开关（无需学习），彻底替代了传统 MoE 的门控网络 $\text{Softmax}(W_{route} x)$。

### 🎭 零号纪元 (Embryonic Phase)：离线人格注入协议

在系统首次部署前，执行离线增量余弦对齐（Delta-Cosine Prediction），在不破坏 Base 的前提下“长出”小说角色心智。

#### 1. 硬件开关与数据准备
* **系统状态：** $\mathbb{I}_{p} = 1$（开启且唯一可训练），$\mathbb{I}_{e} = 0$（关闭）。Base 冻结，海马体 $W$ 关闭。
* **滑动切块：** 将小说文本切分为连续的块 $P_1, P_2, \dots, P_{n+1}$。用冻结的 Base 模型通读，提取每个块末尾的隐藏状态 $H_{P_n}$（离线保存，后续免前向传播）。

#### 2. Delta-Cosine 训练机制
模型被逼迫根据当前的残缺状态，预测下一个状态的**逻辑走向（增量）**。
* **目标增量 (Target Delta)：** 从第 $n$ 幕到第 $n+1$ 幕，语义在隐空间中的矢量位移：
  $$\Delta H = H_{P_{n+1}} - H_{P_n}$$
* **输入构造 (Dropout 强迫思考)：** 模拟真实对话中碎片化的上下文，防止过拟合：
  $$x_{in} = \text{Dropout}(H_{P_n}, p=0.3 \sim 0.5)$$
* **余弦对齐损失函数 (Cosine Alignment Loss)：** 仅要求超高维空间中的“事件发展方向”保持微小夹角，规避 MSE 均值塌陷：
  $$\mathcal{L}_{persona} = 1 - \frac{\text{Expert}_{persona}(x_{in}) \cdot \Delta H}{\|\text{Expert}_{persona}(x_{in})\|_2 \|\Delta H\|_2}$$

**结果：** `Expert_persona` 的权重被永久打上了小说角色的思维烙印。**本阶段结束后，`Expert_persona` 将被转为永久只读（冻结）状态。**

### 🌗 系统的日常生命周期 (The Operational Lifecycle)

由于剥离了 Router，昼夜交替的逻辑变得极其硬核且绝对安全，完美解决了“长期交互导致人格漂移”的致命缺陷。

#### ☀️ 白天模式 (Daytime Inference & Episodic Memory)
* **硬件开关：** $\mathbb{I}_{p} = 1$（冻结），$\mathbb{I}_{e} = 1$（冻结）。
* **海马体状态：** $W$ 激活，按 Delta Rule 高速记录当前对话上下文中的新实体、新指令。
* **整体表现：** 模型同时受 Base（基础智力）、人格轨（小说人物底色）和阅历轨（过往长时记忆）的共同支撑，并用海马体 $W$ 应对当下。任何一个回答都是三重经验与瞬时记忆的完美叠加。

#### 🌙 夜晚模式 (Sleep Consolidation & W-Distillation)
* **核心任务：** 将当天 $W$ 矩阵中高价值的短期记忆，安全地蒸馏进阅历轨，且**绝对不污染原本的人格**。
* **硬件开关：** $\mathbb{I}_{p} = 1$（必须开启，但**绝对冻结**），$\mathbb{I}_{e} = 1$（开启且**唯一可训练**）。
* **物理隔离蒸馏：** 
  * **Teacher：** 走白天的完整通路（Base + 人格轨 + 阅历轨 + 当日演化的 $W$ 轨迹）跑出 logits 分布。
  * **Student：** 关闭 $W$ 矩阵的读取，强迫唯一可训练的 `Expert_episodic` 通过 KL 散度拟合 Teacher。
* **隔离优势：** 无论白天的对话数据分布多么离散或充满噪声，由于 $\mathbb{I}_{p}$ 在数学意义上处于绝对冻结，底色人格毫发无损，只有阅历轨在不断生长。

#### 🌅 翌日重启
1. **归档持久化：** 将夜晚训练好的 `Expert_episodic` 新权重落盘保存。
2. **遗忘清空：** 清零海马体 $W$ 矩阵。
3. **新生重启：** 依然保持 $\mathbb{I}_{p} = 1, \mathbb{I}_{e} = 1$，模型带着更丰满的经历与不变的内核，以清空的工作记忆迎接新的一天。

### 📊 终极版工程建议 (Checklist for Pre-training & Deployment)

| 模块 / 阶段 | 推荐配置 | 架构备注 |
| :--- | :--- | :--- |
| **小说切块 (Chunk Size)** | 128 - 256 tokens | 保持状态演化的微观颗粒度，精准捕捉角色情绪转变。 |
| **输入破坏 (Dropout)** | 0.3 - 0.5 比例 | 极其关键！这是逼迫 `Expert_persona` 进行鲁棒特征提取的核心。 |
| **层级损失权重** | 偏重深层网络 | 浅层多为语法与 Base 特征，深层才是人格语义流向。损失计算向后半部分倾斜。 |
| **防止发散 (L2 约束)** | $\lambda_{reg} \|\Delta \hat{H}\|_2^2$ | 余弦损失只管方向，为防止范数爆炸，加入极弱的 MSE 正则项限制幅度。 |
| **布尔路由开关** | 硬编码在推理图内 | 用系统级 if-else 控制 $\mathbb{I}_p$ 和 $\mathbb{I}_e$，彻底切断梯度回传到错误轨道的可能性。 |
