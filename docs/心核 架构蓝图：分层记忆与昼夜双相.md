## 🌌 核心设计哲学

> **"统一记忆机制,分层时间尺度;白天觉知,夜间巩固。"**

xinhe 的研究假设——智能从"持续状态 + 可塑性 + 多时间尺度"的统一动力系统中涌现——在本架构中落到一个干净的实现:

- **持续状态**:Hippocampus = episode 内演化的 fast weights;Neocortex = 跨 episode 持久的 nn.Parameter(P3 起持久绑定到具体人格)
- **可塑性**:Hippo 通过 TTT inner SGD 实时更新(白天);Neo 通过 outer Adam standard backprop 训练(P-cap 阶段随 LM loss 训;之后 sleep / 人格注入走 learning_session)
- **多时间尺度**:超参数(LR / depth / retention / 可训性开关)直接决定 token 级、chunk 级、book 级、month 级 的频谱

模块化是从两条机制的极值采样(浅 inner SGD vs 深 outer Adam)**涌现**出来的,不是手工设计。

## 🧮 核心数学原理

### 1. Hippocampus 更新规则(NeuralMemory TTT inner SGD)

Hippo MLP fast weights 在 episode 内通过 inner SGD 更新:

$$
W_t = \alpha_t \cdot W_{t-1} - \eta_t \cdot \nabla_W \|\text{MLP}_W(k_t) - v_t\|^2
$$

其中 $\alpha_t \in [0, 1]$ 是 **retention**(每步保留率,对应 Titans paper eq.13 的 `1 - α_t`),$\eta_t$ 是 input-dependent learning rate。
**改用 retention 命名而非 decay**,避免 `decay=0.99` 在数学上是"丢 99%"还是"丢 1%"的语义翻转 bug。

- **`to_decay_factor` input-dependent learnable**(paper Table 5 单一最大贡献项):init bias = `_logit(1 - retention)`,outer Adam 经 backbone backward 学到"什么时候忘什么"
- **`to_adaptive_lr` input-dependent learnable**:init bias = `_logit(base_lr / max_lr)`,模型自行决定每 chunk 多激进
- 推荐 init:Hippo retention=0.99(~100 步半衰期),稳态后实际由模型学到的 `to_decay_factor` 决定
- $\alpha = 0.0$ 等价于每步全清,$\alpha = 1.0$ 等价于不衰减

### 2. Neocortex 更新规则(NeocortexBlock 标准 backprop)

Neo 不走 inner SGD,直接是 nn.Parameter:

$$
y = \text{MLP}_{\text{nn.Parameter}}(\text{RMSNorm}(x))
$$

权重通过 outer Adam 经反向传播更新(P-cap 期间随 LM loss + gate 熵正则联合训练;sleep 时单独 (k,v) 重建 loss + λ_anchor 正则)。
**v9 选择 standard nn.Parameter 而非 NeuralMemory fast weights**:深 MLP 用 inner SGD 严重欠训(test-time SGD 偏爱浅 MLP),改 outer Adam + 多 epoch 是深度学习成熟方法。

**两条路径数学等价(MLP key→value 检索),实现机制故意不同 — 这是 v9 涌现自统一框架的"两点采样"**。

### 3. 两条路径对照

| 维度 | Hippocampus(海马体)| Neocortex(新皮层)|
|---|---|---|
| 类 | `NeuralMemory`(TTT)| `NeocortexBlock`(标准 nn.Module)|
| MLP 形状 | depth=2, expansion=2.0(`d_head→2·d_head→d_head`)| depth=4, expansion=4.0(`d_head→4·d_head→4·d_head→4·d_head→d_head`,每 head 独立 ParameterList)|
| 学习机制 | vmap(grad) 内层 SGD,chunk_size=128 | 普通 forward + outer Adam |
| 学习率 | base_lr=1e-2(input-dep `to_adaptive_lr` 起点)| outer Adam 全局 lr(P-cap=7e-5,sleep 推荐 5e-4)|
| **retention/decay** | init=0.99(~100 步半衰期),`to_decay_factor` learnable | N/A(不是 fast weights)|
| **可训性切换** | `set_daytime_plastic(hippo=...)`(write 是否执行)| `requires_grad_(...)`(是否参与 backprop)|
| **daytime 行为** | True(P-cap + Operational episode 内演化)| P-cap=True(随 LM 走 backprop);Operational=False(冻结) |
| **sleep 行为** | reset_state,不演化(sleep 阶段 1)| backprop 训练(sleep 阶段 2,设计未实施) |
| 状态/参数大小(d_head=64)| ~50KB fast weights / layer / episode | ~400KB nn.Parameter / layer / 持久 |
| 存盘 | episode 内 carry,episode 结束 detach 丢弃 | 与 backbone_addons 同存 ckpt;独立 persona_anchor.pt 接口未实施 |

### 4. 集成 backbone(每个 full_attention 层)

挂载方式:在 backbone hook 6 个 full_attention 层(0.8B 24 层选 6 个,见挂载点章节);hook 在 attention 之后 / FFN 之前。
attention 内的 K/V 已经被 per-layer K/V tokens(P-cap LoRA + persistent prefix)处理过,这里 NM 只在 fresh_mem 前缀位置写入:

```python
class NeuralMemoryPair(nn.Module):
    def forward(self, x, layer_state=None, mem_alpha_override=None):
        # 1. Hippocampus:NeuralMemory(TTT inner SGD,read_before_write=True 内置)
        #    顺序铁律:retrieve 用 episode 入口 weights,再用当前 chunk 写入演化
        h_old = layer_state.hippo if layer_state is not None else None
        r_h, h_new = self.hippocampus(x, state=h_old, read_before_write=True)
        if not self._daytime_plastic_hippo:
            h_new = h_old                                  # 冻结时 h_new 回滚到入口

        # 2. Neocortex:NeocortexBlock(标准 forward,无 state)
        r_n = self.neocortex(x)                            # requires_grad 由外层切换

        # 3. 内容感知 gate(attention-style):让 gate 看见"哪路有内容"
        q = self.gate_q(x)
        scale = 1.0 / math.sqrt(q.shape[-1])
        logit_h = (q * r_h).sum(-1, keepdim=True) * scale
        logit_n = (q * r_n).sum(-1, keepdim=True) * scale
        gate = torch.softmax(torch.cat([logit_h, logit_n], -1), dim=-1)

        # 4. mem_out + RMSNorm(控制范数与 attn 同量级,避免幅值竞争)
        mem_out = gate[..., 0:1] * r_h + gate[..., 1:2] * r_n
        mem_out = self.mem_rmsnorm(mem_out)

        # 5. alpha 开度(reparam 保 alpha ∈ [α_min, 1] 全程可导)
        if mem_alpha_override is not None:
            alpha = torch.tensor(float(mem_alpha_override), device=x.device, dtype=x.dtype)
        else:
            alpha = self.alpha_min + (1.0 - self.alpha_min) * torch.sigmoid(self.alpha_logit)

        x_out = x + alpha * mem_out
        # gate 熵正则在 aux 里 → 进 LM loss 共同优化
        return x_out, LayerMemState(hippo=h_new, neo=None), {"mem_out": mem_out, ...}
```

**MAC 注入(`xinhe_model.py:memory_hook`)**:hook 拿到 `aux["mem_out"]` 后,直接把 `mem_out[:, fresh_start:fresh_end, :]` 加到 hidden 同位置(paper 形态,原 `sigmoid(mac_inject_logit)` gating 实测全程≈1 已删),real token 位置不动 — real 通过 attention 自己 query fresh_mem 取检索结果,paper-faithful。

**关键设计**:
- **head 子空间分离**:NeuralMemory 自带独立 q/k/v projection,Neo 也用独立 multi-head 权重,与 backbone attention head 槽位不竞争
- **read 在 write 之前**(Hippo 内置 `read_before_write=True`):用旧 state 读出 → 再用当前 chunk 更新。反过来会形成自窥,`r_h` 直接看到 `x` 自己,记忆与上下文不可区分
- **内容感知门控**:`gate` 通过 `<q, r_h>` vs `<q, r_n>` 点积决定权重,能感知"哪条路里真有相关内容"——避免冻结 Linear(x) 的字面盲猜
- **alpha 保守 init + reparam**:`alpha = α_min + (1-α_min)·sigmoid(logit)`,init=-5(sigmoid≈0.007)给训练 warmup 缓冲;reparam 保证 sigmoid<α_min 时仍可导(旧版 `clamp(min)` 会断梯度 → alpha_logit 永不学)
- **RMSNorm gate**:控制 mem_out 范数与 attn_out 同量级,避免幅值竞争盖住 backbone
- **`mem_alpha_override` 入口**:`learning_session` 阶段 1 传 `0.0` 走干净 backbone forward,**避免 Hippo 输出污染 Neo 的 (k,v) 教材**(还需配合阶段 1 关 Hippo 的 `daytime_plastic`,否则 write 仍执行)

## 🌗 生命周期:Awake Mode / Learning Mode

### ☀️ Awake Mode(白天聊天)

```
Backbone(含 LoRA):                  frozen ❄️
per-layer K/V tokens:                frozen ❄️
NeuralMemory 静态参数(Hippo):        frozen ❄️
NeocortexBlock 权重:                 🔒 requires_grad=False
gate_q / mem_rmsnorm / alpha_logit:  frozen ❄️
mem_token_init:                      frozen ❄️
Hippocampus fast weights:            ✅ 实时演化(TTT inner SGD,chunk_size=128)
fresh_mem 注入:                      ✅ 每 episode 起 input prefix 注入
跨 turn carry:                       NM weights episode 内连续演化(detach 在 episode 边界)
```

**特性**:对用户可见的行为完全确定、可重现。Neocortex 漂移**数学上不可能**——白天 `requires_grad=False`,梯度无法到达。

### 🌙 Learning Mode(Sleep / 人格注入)── 设计未实施(P2-P4)

**两阶段流程**(待 v9.5 P-cap 验收通过后落实施;v9.5 适配 NeocortexBlock):

```python
def learning_session(self, data_stream, is_persona_injection: bool):
    # ── 阶段 1:Forward 采集干净 (k, v) trajectory ──
    # 关键:走 mem_alpha_override=0 干净 backbone forward,Hippo 通路全关
    # 否则 Hippo 输出会污染 Neo 教材 → 深 Neo 慢慢复刻浅 Hippo → mix_gate 失效

    if is_persona_injection:
        # 注入新人格:Neo 权重重新 init(NeocortexBlock 的 nn.Parameter)
        for w in self.neocortex.weights:
            for h in range(self.n_heads):
                nn.init.xavier_uniform_(w[h])
    else:
        # 日常 sleep:加载 persona_anchor
        self.neocortex.load_state_dict(torch.load("persona_anchor.pt"))
    # Hippo episode 状态清零(sleep 阶段 1 不演化)

    # 显式关 Hippo daytime_plastic:mem_alpha=0 只屏蔽 read 进残差,
    # 但 NeuralMemory 内仍会 update_after_final_store → 浪费计算 + 留污染状态
    self.set_daytime_plastic(hippo=False)
    # Neo:阶段 1 也不参与梯度
    self.neocortex.requires_grad_(False)

    trajectory = []
    with torch.no_grad():
        for chunk in data_stream:
            # mem_alpha_override=0 → 整个 mem 通路屏蔽,backbone 输出干净
            outputs = backbone_forward(chunk, mem_alpha_override=0.0)
            # 取 hook layer 处干净 hidden 作 (k, v) 监督对(具体投影方式 P2 设计时定)
            for layer_idx in hook_layers:
                clean_hidden_l = outputs.hidden_states[layer_idx]
                trajectory.append((layer_idx, clean_hidden_l.detach()))

    # ── 阶段 2:标准 backprop 训练 Neocortex 权重 ──
    self.neocortex.requires_grad_(True)
    optimizer = torch.optim.Adam(self.neocortex.parameters(), lr=5e-4)
    anchor_state = (None if is_persona_injection
                    else {k: v.clone() for k, v in self.neocortex.state_dict().items()})
    for epoch in range(3):
        for ks, vs in DataLoader(trajectory, batch_size=64, shuffle=True):
            pred = self.neocortex(ks)            # NeocortexBlock 标准 forward
            loss = (pred - vs).pow(2).mean()
            if not is_persona_injection:        # sleep 加 anchor 正则
                for k, p in self.neocortex.named_parameters():
                    loss = loss + λ_anchor * (p - anchor_state[k]).pow(2).sum()
            loss.backward()
            optimizer.step()
            optimizer.zero_grad()

    torch.save(self.neocortex.state_dict(), f"neocortex_v{N+1}.pt")
    if is_persona_injection:
        torch.save(self.neocortex.state_dict(), "persona_anchor.pt")  # 永久锚点

    # 恢复 P-cap / Operational 标志
    self.neocortex.requires_grad_(False if mode == "Operational" else True)
    self.set_daytime_plastic(hippo=True)
```

**两阶段分离的根本原因**:
- 阶段 1(forward)**没有梯度**——Neocortex 权重不动,只采集监督对
- 阶段 1 必须走 **`mem_alpha_override=0` 干净路径**——保证 Neo 的 (k,v) 教材未被 Hippo 输出污染,否则深 Neo 会慢慢复刻浅 Hippo,两条记忆通路趋同
- 阶段 2(backprop)**标准 Adam + mini-batch + 多 epoch**——深 MLP 用 inner SGD 严重欠训(test-time SGD 偏爱浅 MLP),这是 v9 把 Neo 改成 NeocortexBlock 的根本动机

## 🛠️ 工程实现要点

### 1. NeuralMemoryPair 类骨架(`xinhe/model/neural_memory_pair.py`)

```python
class NeuralMemoryPair(nn.Module):
    def __init__(
        self,
        d_total: int,
        n_heads: int,
        d_head: int,
        hippo_mlp_depth: int = 2,
        hippo_mlp_expansion: float = 2.0,
        neo_mlp_depth: int = 4,
        neo_mlp_expansion: float = 4.0,
        hippo_retention: float = 0.99,
        hippo_base_lr: float = 1e-2,
        chunk_size: int = 64,
        alpha_logit_init: float = -5.0,
        alpha_min_clamp: float = 0.02,
        phase: str = "P-cap",            # "P-cap" / "Operational"
        gate_entropy_lambda: float = 0.01,
    ):
        super().__init__()
        # Hippo:NeuralMemory(TTT inner SGD,内含 to_decay_factor / to_adaptive_lr 可学)
        self.hippocampus = self._build_neural_memory(
            d_total, n_heads, d_head, hippo_mlp_depth, hippo_mlp_expansion,
            hippo_retention, hippo_base_lr, chunk_size,
        )
        # Neo:NeocortexBlock(普通深 MLP nn.ParameterList per head + AdaptiveRMSNorm)
        self.neocortex = NeocortexBlock(
            d_total, n_heads, d_head,
            depth=neo_mlp_depth, expansion=neo_mlp_expansion,
        )
        # 内容感知 gate:q from x,key 是 r_h / r_n,直接点积
        self.gate_q       = nn.Linear(d_total, d_total, bias=False)
        self.mem_rmsnorm  = AdaptiveRMSNorm(d_total)        # autocast 友好(γ fp32, fwd cast)
        self.alpha_logit  = nn.Parameter(torch.tensor(alpha_logit_init))
        self.alpha_min    = float(alpha_min_clamp)
        self.gate_entropy_lambda = float(gate_entropy_lambda)
        # daytime_plastic flag(只对 Hippo 起作用,Neo 通过 requires_grad_ 控制)
        self._daytime_plastic_hippo = True
        self._daytime_plastic_neo   = (phase == "P-cap")    # API 兼容字段

    def set_daytime_plastic(self, hippo=None, neo=None) -> None: ...
    def forward(self, x, layer_state=None, mem_alpha_override=None) -> tuple[Tensor, LayerMemState, dict]: ...
    # def learning_session(self, data_stream, is_persona_injection): 待 P2 实施
```

### 2. 挂载点策略(Qwen3.5 0.8B)

`xinhe_model.py` hook 6 层 full_attention(24 层中选 6 层),通过 `memory_hook(hidden_states, layer_idx)` 在 attention 之后注入 NM:

- **per-layer K/V**:每个 full_attention 层独立 K/V(N_p=16),在 attention 内拼到 q/k/v 前缀,与 LoRA 联合 fine-tune(P-cap 完成冻结)
- **NM hook**:同样 6 层,从 NeuralMemoryPair 取 mem_out,只写到 fresh_mem 前缀位置

**Qwen3.5 0.8B 是纯 full_attention 24 层**(非 Qwen3-Next hybrid),hook 6 层是计算预算选择,不是因 hybrid 限制。如未来切到 Gated DeltaNet 系列,线性 attention 层不挂 NM(避免"两套 fast weight 抢梯度")。

### 3. mem tokens 插入策略(v9.5 paper-faithful fresh_mem)

**当前实现**:每 episode 入 input prefix 注入 `mem_token_init` 学得的 N_m=16 个 token,序列布局 `[fresh_mem(16), real(T)]`。NM 写入只写到 fresh_mem 段,real 通过因果 attention 自己 query。

**跨 turn carry**:不依赖 KV cache 持久化,而是 **NeuralMemory state.weights episode 内连续演化**(W 在 chunk 边界经 inner SGD 更新,turn 之间不 reset),paper-faithful。

**Episode 边界**:trainer 在 episode 末 `state.detach()`,Hippo fast weights 不跨 episode 持久化(每 episode 重生)。

**长文档模式 fallback**(未实施):若 turn 内 token 数过长,可考虑每 K=128-256 token 注入一组 fresh_mem。当前 turn_max_tokens=768 不需此 fallback。

### 4. 动态扩展(backbone 升级)

所有超参从 backbone config 推:
```python
d_hidden_hippo = hippo_mlp_expansion * backbone.head_dim   # 2.0 * 64 = 128
d_hidden_neo   = neo_mlp_expansion * backbone.head_dim     # 4.0 * 64 = 256
attached_layers = subset_of(backbone.full_attention_layer_indices)   # 0.8B 取 6 / 24
```

| Backbone | d_head | hook 层数 | Neo 权重(单层) | 总 Neo 权重 | 总新增可训练(P-cap) |
|---|---|---|---|---|---|
| Qwen3.5 0.8B(当前)| 64 | 6 / 24 | ~400KB | ~2.5MB | ~50M(含 LoRA + per-layer K/V + Neo + Hippo + mac/mem)|
| Qwen3.5 4B 类(待验)| 128 | ~10 | ~1.6MB | ~16MB | ~500M |
| Qwen3.5 9B 类(待验)| 128 | ~16 | ~1.6MB | ~25MB | ~1B |

**0.8B 验证完跳到 4B/9B 只改 backbone 配置,Neocortex/Hippocampus 自动按比例放大**。
注:如未来切到 Qwen3-Next hybrid(Gated DeltaNet + full_attention 混搭),NM 仅挂 full_attention 层,避免与线性 attention 自带 fast-weight 冲突。

## ⚠️ 已识别风险与防护

| 风险 | 来源 | 防护 |
|---|---|---|
| Neocortex state 容量不足 | 一本小说压成 ~400KB MLP | P3 验收 角色 QA;不达标则 mlp_depth=5 或扩到更多层 |
| 长程 state 数值发散 | 几个月聊天累积演化 | 每天 sleep 后检查 state norm,异常则 anchor 回拉 |
| 人格漂移 | sleep 累积更新 | persona_anchor 永不变 + L2 anchor 正则 + replay buffer |
| Memory 通路整体 collapse | alpha 永不开,mem 完全不用 | `alpha = clamp(σ(logit), 0.02, 1)` 硬下限;P-cap 含强制依赖 mem 的 NIH 任务 |
| **Hippo/Neo 单边偷懒** | 模型只学浅 Hippo,不学深 Neo | ① decay/depth 不对称(物理瓶颈)② P-cap 课程含 >100 turn retention 任务,Hippo 衰减窗口外的答案只能从 Neo 出 ③ Neo 走 outer Adam 标准 backprop(P-cap 自然有梯度,无需"灌水")④ `mix_gate` 熵正则 `L += -λ·H(gate), λ=0.01` ⑤ 监控 `H(gate) > 0.4` ⑥ sleep 阶段 1 关 Hippo plastic + `mem_alpha_override=0` 干净路径采集 (k,v) 教材(P2 实施) |
| backbone attention 与 mem 幅值竞争 | mem_out 范数过大 | RMSNorm + alpha 保守 init + backbone 冻结兜底 |
| 灾难性遗忘(早期 sleep 数据)| 历史 replay buffer 退化 | 水库采样 + 显式保留早期"金标准 trajectory" |
| **Neo 教材被 Hippo 污染** | learning_session 阶段 1 走正常 forward 时,backbone 输出已混入 `α·r_h`,Neo 据此提的 (k,v) 不干净;深 Neo 慢慢复刻浅 Hippo,两路趋同 | 阶段 1 双重防护:`mem_alpha=0.0` 屏蔽残差 + 显式关 `daytime_plastic` 阻止隐式 write;阶段 2 完恢复 |
| **read/write 顺序错位形成自窥** | forward 里若先 write 再 read,当前 chunk 自己的信息立刻进入 `r_h`,记忆与上下文混淆,门控失去判别意义 | 铁律:`r = mem.read(x)` 先于 `mem.write(x)`,Hippo / Neo 对称 |
| **P-cap 期间 mix_gate 把 Neo 学死** | P-cap 阶段 Neo 输出 $r_n$≈0 → gate 学成 `[1, 0]` 永久无视 Neo;后续 P3 注入再用力都白搭 | v9 通过两条机制达成:① Neo = NeocortexBlock(nn.Parameter)P-cap `requires_grad=True`,outer Adam 自然有梯度,$r_n$ 随训练增长;② `λ_gate_entropy=0.01` 熵正则强制双边参与;监控 `H(gate) > 0.4` 不达标则上调 λ |

## 🌳 设计哲学的最终落地

```
   持续状态        ← Hippocampus fast weights(episode 内) + Neocortex nn.Parameter(持久)
   可塑性          ← Hippo TTT inner SGD(白天) + Neo standard backprop(白天 P-cap / 夜间 sleep)
   多时间尺度      ← LR / retention / 可训性开关 → token 级 / book 级 / month 级
   从两条路径涌现  ← MLP 检索数学等价,实现机制故意不同(浅 inner SGD vs 深 outer Adam)
```

**两条路径,数学等价(MLP key→value),实现互补(inner SGD 偏浅 / backprop 偏深),昼夜双相。**
**无手工模块化拼接。无双流 routing。Cortex 全秩(NeocortexBlock 不用 LoRA);backbone 在 P-cap 用 LoRA 做轻量适配。**

记忆从统一动力系统中涌现 — 同一个"MLP 当 key→value 检索器"原则在两个超参极值采样下,自然分化为浅快 / 深慢两条路径。
