## 🌌 核心设计哲学

> **"统一记忆机制,分层时间尺度;白天觉知,夜间巩固。"**

xinhe 的研究假设——智能从"持续状态 + 可塑性 + 多时间尺度"的统一动力系统中涌现——在本架构中落到一个干净的实现:

- **持续状态**:Hippocampus 与 Neocortex 都是持续演化的 fast weights(MLP 权重 = 记忆状态)
- **可塑性**:Hippocampus 通过 test-time SGD 实时更新;Neocortex 通过 sleep-time 标准 backprop 训练
- **多时间尺度**:超参数(LR / depth / 可训性开关)直接决定 token 级、chunk 级、book 级、month 级 的频谱

模块化是从超参数空间的两点采样**涌现**出来的,不是手工设计。

## 🏛️ 三层架构

### Layer 1:能力(Capability)── 一次训练,永久冻结

| 组件 | 形态 | 训练时机 |
|---|---|---|
| Qwen3-Next backbone | 原 base 权重 | 永远不动 |
| persistent_memory tokens | 每 full-attn 层 `(2, H, N=32, d_head)` | P-cap |
| NeuralMemory 静态参数 | `q_proj / k_proj / v_proj / combine_heads / mix_gate / alpha_logit` | P-cap |
| mem tokens 初始 embedding | `(N_mem, d_hidden)` 学出的"空白笔记本页"初值 | P-cap |

**学的是**:"如何使用记忆模块",不是任何具体内容。
**训完**:全部冻结,作为后续所有人格的共同底座。

### Layer 2:人格(Persona)── 每本小说一份,可热替换

| 组件 | 形态 | 何时生成 |
|---|---|---|
| Hippocampus state | 浅 MLP fast weights(~50KB / 0.8B base) | 每 session 重生 |
| Neocortex state | 深 MLP fast weights(~400KB / 0.8B base) | 人格注入时生成 |
| `persona_anchor.pt` | Neocortex state 在人格注入完成时的快照 | 注入完成后存盘,**永不覆盖** |

**存的是**:角色的认知、世界观、说话风格。
**切换人格**:加载不同 state 文件,毫秒级。
**多人格**:同一份 Capability 配多个 persona state 文件。

### Layer 3:操作(Operational)── 日常使用

| 组件 | 形态 | 行为 |
|---|---|---|
| Hippocampus state | fast weights | 白天实时演化(test-time SGD,1 步)|
| Neocortex state | fast weights | 白天 frozen 只读;sleep 时打开训练 |
| mem tokens KV cache | per-turn `(K, V)` 张量 | session 内跨 turn 持久化,session 结束清 |

## 🧮 核心数学原理

### 1. NeuralMemory 统一更新规则

所有"记忆"都通过 MLP fast weights 通过 SGD 更新存储:

$$
W_t = \alpha \cdot W_{t-1} - \eta \cdot \nabla_W \|\text{MLP}_W(k_t) - v_t\|^2
$$

其中 $\alpha \in [0, 1]$ 是 **retention**(每步保留率,对应 Titans paper eq.13 的 `1 - α_t`)。
**改用 retention 命名而非 decay**,避免 `decay=0.99` 在数学上是"丢 99%"还是"丢 1%"的语义翻转 bug。

- $\alpha = 1.0$:不衰减(Neocortex,靠 anchor 正则约束漂移)
- $\alpha = 0.99$:每步保留 99%,~100 步半衰期(Hippocampus 自然遗忘)
- $\alpha = 0.0$:每步全清(等价于无记忆 baseline)

**Hippocampus 与 Neocortex 共享同一类、同一更新方程,只差超参**。

### 2. 两个实例化对照

| 超参 | Hippocampus(海马体) | Neocortex(新皮层) |
|---|---|---|
| MLP 形状 | `d_head → 2·d_head → d_head`(depth=2)| `d_head → 4·d_head → 4·d_head → 4·d_head → d_head`(depth=4)|
| 基础 LR | 1e-2(快学) | 5e-4(慢学,sleep 用) |
| **retention** $\alpha$ | 0.99(~100 步半衰期,自然遗忘)| 1.0(不衰减,靠 anchor 正则约束漂移) |
| **daytime_plastic** | **True**(test-time SGD)| **False**(冻结) |
| **sleep_trainable** | False(默认不参与)| **True**(标准 backprop)|
| 内迭代步数 | 1(浅 MLP 单步够)| K=3 epoch × batch(深 MLP 需充分收敛)|
| state 大小(d_head=64)| ~50KB | ~400KB |
| 存盘策略 | session 结束丢弃 | persona_anchor + nightly versioned save |

### 3. 集成 backbone(每个 full-attn 层)

```python
def forward(self, x, mode, mem_alpha_override=None):
    attn_out = self.attn(x)  # 含 persistent_memory 拼到 k/v 前缀(冻结)
    
    # **顺序铁律:先 read 旧状态,再 write 当前 chunk**
    # 反过来会让当前 chunk 自己的信息立刻进入读出,形成"自窥"回路
    
    # Hippocampus:awake 时实时演化
    r_h = self.hippo.read(x)              # 用旧 state 读出(B, T, d_total)
    if self.hippo.daytime_plastic:
        self.hippo.write(x)               # 再用当前 chunk 更新
    
    # Neocortex:Operational 严格只读;P-cap 临时开 plastic(防 gate 学死,见下节)
    # plastic=True 时 read/write 同样要保证"先读后写"
    r_n = self.neo.read(x)
    if self.neo.daytime_plastic:
        self.neo.write(x)
    
    # ── 内容感知门控(attention-style)──
    # x 作 query,r_h / r_n 作 key,直接点积取置信度 → softmax
    # 核心:让 gate 看见"哪一路真有内容、哪一路恒为零",而不是从 x 字面瞎猜
    q = self.gate_q(x)  # (B, T, d_total)
    logit_h = (q * r_h).sum(-1, keepdim=True) / math.sqrt(q.shape[-1])
    logit_n = (q * r_n).sum(-1, keepdim=True) / math.sqrt(q.shape[-1])
    gate = torch.softmax(torch.cat([logit_h, logit_n], -1), dim=-1)
    
    mem_out = gate[..., 0:1] * r_h + gate[..., 1:2] * r_n
    mem_out = self.mem_rmsnorm(mem_out)  # 控制范数与 attn_out 同量级
    
    # alpha:可学开度 + 硬下限防 collapse + sleep 阶段 1 可外部覆写为 0(干净路径)
    alpha = self.alpha_logit.sigmoid().clamp(min=0.02)
    if mem_alpha_override is not None:
        alpha = mem_alpha_override
    
    return x + attn_out + alpha * mem_out
```

**关键设计**:
- **head 子空间分离**:NeuralMemory 自带独立 q/k/v projection,与 backbone attention 在不同 head 子空间,**不存在 head 槽位竞争**
- **read 在 write 之前**:用旧 state 读出 → 再用当前 chunk 更新。反过来会形成自窥,`r_h` 直接看到 `x` 自己,记忆与上下文不可区分
- **内容感知门控**:`gate` 通过 `<q, r_h>` vs `<q, r_n>` 点积决定权重,能感知"哪条路里真有相关内容"——避免冻结 Linear(x) 的字面盲猜
- **alpha 保守 init + clamp 下限**:训练初期 mem 通路贡献 < 1%,backbone 行为几乎不变,梯度逐步开放;`clamp(0.02, 1)` 防完全 collapse
- **RMSNorm gate**:控制 mem_out 范数与 attn_out 同量级,避免幅值竞争盖住 backbone
- **`mem_alpha_override` 入口**:`learning_session` 阶段 1 传 `0.0` 走干净 backbone forward,**避免 Hippo 输出污染 Neo 的 (k,v) 教材**(还需配合阶段 1 关 `daytime_plastic`,否则 write 仍执行)

## 🌗 生命周期:Awake Mode / Learning Mode

### ☀️ Awake Mode(白天聊天)

```
Backbone:               frozen ❄️
persistent_memory:      frozen ❄️
NeuralMemory 静态参数:   frozen ❄️
Hippocampus state:      ✅ 实时演化(test-time SGD,1 步/chunk)
Neocortex state:        🔒 frozen,只读
mix_gate / alpha:       frozen ❄️
mem tokens KV cache:    ✅ 跨 turn 累积
```

**特性**:对用户可见的行为完全确定、可重现。Neocortex 漂移**数学上不可能**——白天不打开它的可训性开关,梯度无法到达。

### 🌙 Learning Mode(Sleep / 人格注入)

**两阶段流程,完全统一**:

```python
def learning_session(data_stream, is_persona_injection: bool):
    # ── 阶段 1:Forward 采集干净 (k, v) trajectory ──
    # 关键:走 mem_alpha=0 干净 backbone forward,Hippo 通路全关
    # 否则 Hippo 输出会污染 Neo 教材 → 深 Neo 慢慢复刻浅 Hippo → mix_gate 失效
    # sleep 数据非流式聊天,Hippo 演化无价值,直接关闭
    
    if is_persona_injection:
        neocortex.reset_state()        # 注入新人格,从空开始
    else:
        neocortex.load(persona_anchor) # 日常 sleep,从人格锚点起步
    hippocampus.reset_state()           # sleep 阶段 1 Hippo 不演化
    
    # **显式关 daytime_plastic**:仅 reset_state 不够
    # mem_alpha=0 只屏蔽 read 进残差,但 forward 里 `if plastic: write(x)` 还会被调用
    # → 浪费计算 + 阶段 1 结束后留下污染状态
    saved_flags = (hippocampus.daytime_plastic, neocortex.daytime_plastic)
    hippocampus.daytime_plastic = False
    neocortex.daytime_plastic   = False
    
    trajectory = []
    with torch.no_grad():
        for chunk in data_stream:
            # mem_alpha=0 → 整个 mem 通路被屏蔽,backbone 输出干净
            clean_hidden = backbone(chunk, mem_alpha=0.0)
            k = neocortex.k_proj(clean_hidden)
            v = neocortex.v_proj(clean_hidden)
            trajectory.append((k.detach(), v.detach()))
    
    # 阶段 1 结束恢复(stage 2 走标准 backprop,不依赖 daytime_plastic)
    hippocampus.daytime_plastic, neocortex.daytime_plastic = saved_flags
    
    # ── 阶段 2:标准 backprop 训练 Neocortex state ──
    optimizer = torch.optim.Adam([neocortex.state], lr=5e-4)
    for epoch in range(3):
        for ks, vs in DataLoader(trajectory, batch_size=64, shuffle=True):
            pred = neocortex.mlp(ks, weights=neocortex.state)
            loss = (pred - vs).pow(2).mean()
            
            # Sleep 时加 anchor 正则,防漂移;人格注入时不加
            if not is_persona_injection:
                loss += λ_anchor * (neocortex.state - persona_anchor).pow(2).sum()
            
            loss.backward()
            optimizer.step()
            optimizer.zero_grad()
    
    save(neocortex.state, f"neocortex_v{N+1}.pt")
    if is_persona_injection:
        save(neocortex.state, "persona_anchor.pt")  # 永久锚点
```

**两阶段分离的根本原因**:
- 阶段 1(forward)**没有梯度**——Neocortex state 不动,只采集数据
- 阶段 1 必须走 **`mem_alpha=0` 干净路径**——保证 Neo 的 (k,v) 教材未被 Hippo 输出污染,否则深 Neo 会慢慢复刻浅 Hippo,两条记忆通路趋同
- 阶段 2(backprop)**用标准 Adam + mini-batch + 多 epoch**——这是深度学习里训练深 MLP 的成熟方法
- **不再依赖 test-time SGD 训深 MLP**(test-time SGD 偏爱浅 MLP,深 MLP 用它是欠训练的)

### 🌅 翌日重启

1. 加载昨夜训完的新版 Neocortex state(`neocortex_v{N}.pt`)
2. Hippocampus state 清零
3. mem tokens KV cache 清空
4. `persona_anchor.pt` **永远不变**——这是防长期漂移的根基

## 🎓 训练范式

### P-cap:能力训练(一次性,2-3 周,0.8B base 上 1-3B tokens)

**数据**(diverse long-context,**不是任何特定 persona**):

| 类型 | 占比 | 来源 | 作用 |
|---|---|---|---|
| 多本小说 / 长篇文学(混杂题材)| 35% | RedPajama / Books / PG-19 | 跨题材吸收叙事 |
| 长对话(多轮、跨 turn 引用)| 25% | UltraChat / ShareGPT | 学跨 turn retrieval |
| 技术文档 / 论文 | 15% | arXiv / wiki | 事实型 retrieval |
| 合成 NIH 任务 | 15% | 自动生成 | 显式监督精准 retrieval |
| 通用 instruct chat | 10% | Alpaca / Tulu | 防能力崩 |

**训练目标**:标准 LM loss + NIH 位置加权 + **mix_gate 熵正则 `λ=0.01`**(防单边塌缩)
**冻结**:Qwen3 backbone 全冻
**可训练**:NeuralMemory 静态参数(q/k/v/combine_heads)+ persistent_memory + mem tokens 初值 + `gate_q` + `alpha_logit`

**特殊设置 — Neocortex 不能缺席**:
- **P-cap 期间 Neocortex 也临时启用 `daytime_plastic=True`,lr=1e-4**(远小于 Hippo 1e-2)
- **理由**:若 Neo state 全程 ≈ 0,$r_n$ 恒为零,`mix_gate` 会学成 `[1, 0]` **永久无视 Neo**,后续 P3 注入再用力都白搭
- **小 lr 的目的**:让 Neo state 持续有非零内容供 gate 感知,但不让它真去吸收 P-cap 数据(那是 P3 的事)
- **P-cap 完成后**:Neocortex state **弃用**(P3 从空 reset 重新注入人格);Hippocampus state **弃用**(session 重生)。只保留静态参数 + 静态 mem tokens

**验收**:LongBench / NIH 准确率达 baseline、通用 chat MMLU/ARC ≥ Base × 0.95、**`H(mix_gate) > 0.4`**

### P3:人格注入(每本小说 ~1 小时)

- **数据**:单本小说原文,**无任何增强**
- **流程**:`learning_session(novel_chunks, is_persona_injection=True)`
- **输出**:`hippocampus_state.pt`(可丢) + `neocortex_state.pt`(同时存为 `persona_anchor.pt`)
- **验收**:角色 QA(novel 内)≥ 80%、角色 RP(novel 外)人评 ≥ 7/10

### P4:日常 sleep(每天 / 累积 N 天触发)

- **数据**:今日聊天 trajectory(60%) + 人格 replay(25%) + 历史 replay(15%)
- **流程**:`learning_session(mixed_data, is_persona_injection=False)`
- **触发策略**:
  - 累积型:今日 token 数 < 10K → 入 pending,累积 ≥ 50K 触发一次合并 sleep
  - 防漂移:λ_anchor 5e-4 ~ 1e-3,从严起步
- **输出**:`neocortex_v{N+1}.pt`(版本化保留)

## 🛠️ 工程实现要点

### 1. NeuralMemoryPair 类骨架

```python
class NeuralMemoryPair(nn.Module):
    def __init__(self, d_head, n_heads, phase="P-cap"):
        d_total = d_head * n_heads
        # daytime_plastic 在 P-cap / Operational 不同
        # P-cap:Neo 也开 plastic(lr 小),防 mix_gate 学死 Neo
        # Operational:Neo 严格冻结,只在 sleep 打开
        neo_plastic = (phase == "P-cap")
        neo_lr      = 1e-4 if phase == "P-cap" else 5e-4  # P-cap 极小 / sleep 标准
        
        self.hippocampus = NeuralMemory(
            d_head=d_head, n_heads=n_heads,
            mlp_depth=2, mlp_hidden=2 * d_head,
            base_lr=1e-2, retention=0.99,         # 每步保留 99%
            daytime_plastic=True, sleep_trainable=False,
        )
        self.neocortex = NeuralMemory(
            d_head=d_head, n_heads=n_heads,
            mlp_depth=4, mlp_hidden=4 * d_head,
            base_lr=neo_lr, retention=1.0,        # 不衰减(靠 anchor 正则)
            daytime_plastic=neo_plastic, sleep_trainable=True,
        )
        # 内容感知门控:q from x,key 是 r_h / r_n,直接点积
        self.gate_q       = nn.Linear(d_total, d_total, bias=False)
        self.mem_rmsnorm  = MultiheadRMSNorm(d_head, n_heads)
        self.alpha_logit  = nn.Parameter(torch.tensor(-5.0))
    
    def forward(self, x, mode, mem_alpha_override=None):
        ...
    
    def learning_session(self, data_stream, is_persona_injection):
        ...
```

### 2. 挂载点策略(Qwen3-Next hybrid)

仅挂在 **full-attn 层**:
- 0.8B:~6 个 full-attn 层
- 4B:~10 个
- 9B:~16 个

**线性 attention 层(Gated DeltaNet)不挂**——它们自带 fast-weight 状态,加 NeuralMemory 是冗余,会触发"两套 fast weight 抢梯度"的 v6 自锁回路。

### 3. mem tokens 插入策略

**聊天模式**(主):turn 边界(`<|im_end|>` 之后)插 N=8-16 个

**长文档模式**:每 K=128-256 个 token 强制插一组(无 turn 边界时 fallback)

**KV 持久化**:session 内跨 turn 保留(打破 vanilla Titans 的 block-diagonal mask)→ 后续 turn 真实 token 都能 attend 到前面所有 turn 的 mem 摘要

### 4. 动态扩展(backbone 升级)

所有超参从 backbone config 推:
```python
d_hidden_hippo = 2 * backbone.head_dim
d_hidden_neo   = 4 * backbone.head_dim
attached_layers = backbone.full_attn_layer_indices
```

| Backbone | d_head | n_full_attn | Neo state(单层) | 总 state | 总新增可训练参数 |
|---|---|---|---|---|---|
| Qwen3-Next 0.8B | 64 | ~6 | ~400KB | ~2.5MB | ~80M |
| Qwen3-Next 4B | 128 | ~10 | ~1.6MB | ~16MB | ~500M |
| Qwen3-Next 9B | 128 | ~16 | ~1.6MB | ~25MB | ~1B |

**0.8B 验证完跳到 4B/9B 只改 backbone 配置,Neocortex/Hippocampus 自动按比例放大**。

## ⚠️ 已识别风险与防护

| 风险 | 来源 | 防护 |
|---|---|---|
| Neocortex state 容量不足 | 一本小说压成 ~400KB MLP | P3 验收 角色 QA;不达标则 mlp_depth=5 或扩到更多层 |
| 长程 state 数值发散 | 几个月聊天累积演化 | 每天 sleep 后检查 state norm,异常则 anchor 回拉 |
| 人格漂移 | sleep 累积更新 | persona_anchor 永不变 + L2 anchor 正则 + replay buffer |
| Memory 通路整体 collapse | alpha 永不开,mem 完全不用 | `alpha = clamp(σ(logit), 0.02, 1)` 硬下限;P-cap 含强制依赖 mem 的 NIH 任务 |
| **Hippo/Neo 单边偷懒** | 模型只学浅 Hippo,不学深 Neo | ① decay/depth 不对称(物理瓶颈)② P-cap 课程含 >100 turn retention 任务,Hippo 衰减窗口外的答案只能从 Neo 出 ③ sleep 时 detach Hippo 路径,梯度专给 Neo ④ `mix_gate` 熵正则 `L += -λ·H(gate), λ=0.01` ⑤ 监控 `H(gate) > 0.4` |
| backbone attention 与 mem 幅值竞争 | mem_out 范数过大 | RMSNorm + alpha 保守 init + backbone 冻结兜底 |
| 灾难性遗忘(早期 sleep 数据)| 历史 replay buffer 退化 | 水库采样 + 显式保留早期"金标准 trajectory" |
| **Neo 教材被 Hippo 污染** | learning_session 阶段 1 走正常 forward 时,backbone 输出已混入 `α·r_h`,Neo 据此提的 (k,v) 不干净;深 Neo 慢慢复刻浅 Hippo,两路趋同 | 阶段 1 双重防护:`mem_alpha=0.0` 屏蔽残差 + 显式关 `daytime_plastic` 阻止隐式 write;阶段 2 完恢复 |
| **read/write 顺序错位形成自窥** | forward 里若先 write 再 read,当前 chunk 自己的信息立刻进入 `r_h`,记忆与上下文混淆,门控失去判别意义 | 铁律:`r = mem.read(x)` 先于 `mem.write(x)`,Hippo / Neo 对称 |
| **P-cap 期间 mix_gate 把 Neo 学死** | P-cap 阶段 Neo state≈0 → $r_n$≈0 → gate 学成 `[1, 0]` 永久无视 Neo;后续 P3 注入再用力都白搭 | P-cap 期间 Neo 临时 `daytime_plastic=True`,lr=1e-4 远小于 Hippo,让 Neo state 持续有非零内容供 gate 感知;P-cap 完成后 state 弃用 |

## 🛣️ Phase 实施路线

```
P-cap   能力训练 ── Hippo + Neo 静态参数 + persistent + mix_gate    2-3 周
P0      集成 NeuralMemoryPair,替换 v8 的 Hippocampus W            3 天
P1      mem tokens turn 边界插入 + KV 跨 turn 持久化               5 天
P2      state save/load + Neocortex learning_session 接口          3 天
P3      人格注入(读小说 → 标准 backprop 训 Neocortex → 存 anchor) 2 天
P4      日常 sleep(每天/每 N 天 → train Neocortex)                3 天
P5      连续 7 天聊天 + state 漂移监控 + anchor drift 校准         1 周
```

每个 phase 都能**独立验收 + 独立回滚**。P0-P2 是工程,稳;P-cap、P3-P5 是研究,有不确定性。

## 🌳 设计哲学的最终落地

```
   持续状态        ← Hippocampus state + Neocortex state(MLP fast weights)
   可塑性          ← test-time SGD(白天)+ standard backprop(夜间)
   多时间尺度      ← LR / decay / 可训性开关 → token 级 / book 级 / month 级
   从统一中涌现    ← 同一个 NeuralMemory 类,两个超参实例
```

**两个 MLP,一种机制,昼夜双相。**
**无手工模块化拼接。无双流 routing。无 LoRA 低秩约束。**

记忆从统一动力系统中涌现,而不是被设计出来。
